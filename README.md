# Notebook: Mercado Financeiro para Iniciantes 📈

Este repositório contém o acesso e as fontes de estudo para um guia abrangente sobre o ecossistema do mercado financeiro.

## 🚀 Acesse o conteúdo interativo
O projeto foi desenvolvido utilizando o **NotebookLM**, onde você pode interagir com as fontes, fazer perguntas à IA e explorar os conceitos de forma dinâmica.

## 📋 Conteúdo abordado
O texto oferece uma visão detalhada sobre:
*   **Fundamentos:** O que é e como funciona o ecossistema financeiro.
*   **Dinâmica de Mercado:** Interação entre agentes econômicos.
*   **Guia para Iniciantes:** Passos essenciais para quem está começando a estudar investimentos.
